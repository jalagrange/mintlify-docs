# Quick Start Guide

This guide helps developers understand and work with the Dynamic Quiz Generator system.

<Callout type="info">
  **New to the project?** Start with the [Project Overview](/project-overview) to understand what we're building and why.
</Callout>

## Prerequisites

- Node.js 18+
- Understanding of REST APIs
- Familiarity with AI/ML concepts

## Development Setup

<AccordionGroup>
  <Accordion icon="github" title="Clone the repository">
    Clone the Dynamic Quiz Generator repository to your local machine:
    ```bash
    git clone https://github.com/go1/dynamic-quiz-creator.git
    cd dynamic-quiz-creator
    ```
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Preview documentation changes">
    To work on documentation locally:
    1. Install the Mintlify CLI globally:
    ```bash
    npm i -g mintlify
    ```
    2. Navigate to the docs directory and start the dev server:
    ```bash
    cd mintlify-docs
    mintlify dev
    ```
    3. Open [http://localhost:3000](http://localhost:3000) to view the documentation
  </Accordion>
</AccordionGroup>
